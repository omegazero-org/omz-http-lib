## omz-http-lib/net

Contains integrations with [omz-net-lib](https://git.omegazero.org/omegazero/omz-net-lib).

Requires *http-core*, [omz-java-lib](https://git.omegazero.org/omegazero/omz-java-lib) and [omz-net-lib](https://git.omegazero.org/omegazero/omz-net-lib).

Prebuilt JARs: <https://drone.omegazero.org/build-artifacts/java/org.omegazero.http:omz-http-net>
